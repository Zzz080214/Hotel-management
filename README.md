# 酒店管理系统

这是一个酒店管理项目，包含 Spring Boot 后端、Vue 管理后台、微信小程序和数据库脚本。

## 目录

- `backend/`: Spring Boot API 服务
- `vue-admin/`: Vue 管理后台
- `miniprogram/`: 微信小程序
- `database/`: 数据库初始化脚本和说明

## 本地配置

后端数据库连接支持通过环境变量配置：

- `DB_URL`
- `DB_USERNAME`
- `DB_PASSWORD`
- `WX_MINI_PROGRAM_SECRET`

测试环境可使用：

- `DB_TEST_URL`
- `DB_TEST_USERNAME`
- `DB_TEST_PASSWORD`
